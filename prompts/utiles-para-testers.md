# Utiles para testers

## Este documento contiene prompts utiles para realizar el flujo de pruebas manuales y automatizadas.

### 1.Generar casos de prueba para un ticket de Jira
Genera casos de prueba para el ticket PG-xxxx. 

### 2.Ejecutar caso de prueba y generar evidencia
Ejecuta el caso de prueba TC-xxx del archivo pg-xxx.md

### 3.Agregar casos de prueba al ticket de Jira

comenta en pg-xxxx con la lista (solo titulos) de los casos de prueba generados en el archivo pg-xxxx.md

### 4.Agregar comentario con evidencia al ticket

Comenta en PG-xxxx el resultado del TC-xxx
